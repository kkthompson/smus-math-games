# SMUS Math Games

A collection of browser-based math games for Grade 4, built for fun and speed.

## Games

### [Times Table Blitz!](times-table/)
Race the clock through 10 multiplication questions. Choose your table (2–12 or mixed), pick a difficulty, and see how high you can score.

### [Divide & Conquer!](division/)
Test your division skills under pressure. Same format as Blitz — 10 questions, multiple difficulties, and a leaderboard to top.

## Difficulties

| Level | Description |
|-------|-------------|
| Easy | 6 answer choices from the same table family |
| Medium | 6 choices including tricky near-miss numbers |
| Hard | Near-miss distractors + reversed questions (e.g. `? × 5 = 35`) |
| Extra Hard | 9 choices, shuffled — all the Hard tricks, no order to help you |

## Running Locally

Just open `index.html` in a browser — no build step or server required.
